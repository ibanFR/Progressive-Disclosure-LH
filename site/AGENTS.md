# Site AGENTS instructions

This is the **published** Jekyll site (theme: `just-the-docs`) for the Progressive Disclosure Learning Hour. These are public-facing pages, deployed to GitHub Pages via `.github/workflows/pages.yml`.

## Replies

**ALWAYS** start replies with 📄. Stack emojis to show which harnesses you are using.

## Pages

`index.md` is the site home (the Learning Hour session plan) — not a Diataxis type.

The rest follow the Diataxis model — one type per page, never mixed:

- `tutorial/` — learning-oriented lessons for a newcomer.
- `how-to/` — task-oriented guides for a user with a goal.
- `reference/` — dry, exhaustive, lookup-oriented facts.
- `explanation/` — understanding-oriented discussion and background.

## Harnesses

- Page content is **NEVER** handwritten or hand-edited. **ALWAYS** read
  [harness/create-or-update-page.md](harness/create-or-update-page.md) and follow it
  **BEFORE** creating or editing any page.
- Read [harness/preview-site.md](harness/preview-site.md) when previewing the site locally.
